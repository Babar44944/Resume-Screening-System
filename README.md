# 🤖 AI-Powered Resume Screening System

## 🎯 Project Overview

The **AI-Powered Resume Screening System** is an intelligent recruitment solution developed as part of an AI/ML Internship project.

The system leverages **Natural Language Processing (NLP)** and **Machine Learning** techniques to automatically analyze resumes, extract relevant information, classify job categories, and predict candidate suitability for specific job roles.

This project aims to reduce manual screening efforts and improve the efficiency of the hiring process through AI-driven decision-making.

---

## 🚀 Features

✅ Resume Analysis & Processing

✅ Skill Extraction using NLP

✅ Job Category Classification

✅ Candidate Suitability Prediction

✅ Interactive Web Interface with Streamlit

✅ Data Visualization & Insights

✅ Automated Resume Screening Workflow

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/Babar44944/Resume-Screening-System.git
cd Resume-Screening-System
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

---

## ▶️ Run the Application

### Option 1 automatically everything install

```bash
python run.py
```
